Extract a number represented as a C++ type T from a vector representing a buffer in memory.

Template Parameters
TA C++ type such as int, unsigned int, long long, unsigned long long, float and double
Parameters
dataBufferA vector of chars representing a buffer in memory
offsetStart position on the buffer where the function perform the extraction
sizeSize of the data to be extracted to convert into a number of type T
Note
Be carefull with the size you provide. If the size and the C++ type doesn't match the function throw a length_error exception
Exceptions
std::out_of_rangewhen extract with parameters produce extraction out of vector range.
std::length_errorwhen C++ data type size provided in T does not match with size parameter
Returns
T The number extracted represented as the C++ type provided by T
